地图制作器 Beta 1.2.0
适用版本 Beta 1.0.0+
By Fallen_Breath

菜单↑↓选择，回车确定

制作器
WASD   : 移动
空格键 : 放置，重复按下可更改方向(模式：方块)
退格键 : 删除
←→   : 选择方块/复活点
↑↓   : 选择模式(方块/复活点)
数字键 : 更改方块附加值(模式：方块)
         选择复活点(模式：复活点)
-      : 使方块附加值取相反数(模式：方块)
O      : 查看方块/复活点/方块附加值
X      : 进入详细设置
F1     : 保存
F2     : 开启/关闭自动放置
F3     : 开启/关闭自动删除
Esc    : 保存并退出

详细设置
↑↓   : 选择
回车   : 输入/确定
